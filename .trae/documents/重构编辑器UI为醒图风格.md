## 顶部栏
- 左侧：`返回(X)` 与 `帮助(?)`，采用深色背景与高对比度图标
- 中间：`一键超清` 下拉菜单，支持预设：原图、降噪、锐化、增强（后续可接开源/本地算子）
- 右侧：`导出` 主按钮（亮绿色），统一走当前模式的保存流程
- 支持长按预览进入“原图对比”态（按住显示原图、松手恢复）

## 画布区域
- 居中显示图片，`ContentScale.Fit`，黑色背景
- 统一处理旋转与硬件位图转软位图，沿用现有 `loadCompressedBitmap`
- 预览使用 1080p 小图，保存使用原图（双位图策略）

## 底部导航（一级分类）
- 分类列表：`模板 / 人像 / 滤镜 / 调节 / 贴纸 / 文字 / 特效`
- 使用 `LazyRow`，当前选中以亮绿色文字高亮，未选中灰色
- 切换分类时，更新二级工具面板（见下）

## 二级工具面板（随一级分类变化）
- 人像：`一键美化(NEW) / 面部重塑 / 瘦脸瘦身(NEW) / 面部丰盈 / 自动美颜`
- 滤镜：基础色彩滤镜（暖色/冷色/黑白等），后续支持 LUT/曲线
- 调节：`亮度 / 对比度 / 饱和度` 滑块
- 拼图在“模板”分类下提供入口（进入拼图编辑器）
- 贴纸/文字/特效：先占位，提供基础结构与示例项
- 工具项以图标+标签展示，`NEW` 角标使用小绿标覆盖

## 交互规则
- 点击一级分类：高亮并刷新工具面板
- 点击二级工具：顶部下方显示对应“控制面板”（滑块或网格选择）
- 控制面板调整实时更新预览（IO 线程处理，主线程展示）
- `导出` 根据当前分类执行：人像与滤镜输出处理后的位图，拼图渲染 `View` 为位图

## 架构与数据模型
- `EditorMode`：`FILTER / COLLAGE / BEAUTY`（默认进入与用户操作对应）
- `ToolCategory(id, name, icon)`；`ToolItem(id, name, badge)`；`EditorState(selectedCategory, selectedTool)`
- 顶部栏/底部栏/工具面板拆分为独立 Composable，`EditorShell` 作为容器

## 实现分解
1. 新建 `EditorShell`：包含顶部栏、画布、底部分类与工具面板
