## 目标

* 在上半区与底部功能网格之间填充一张“英雄海报卡”，提升品牌感与视觉平衡，同时提供快速入口。

* 保持艺术简洁，不引入复杂动画与滚动干扰。

## 英雄卡内容

* 结构：圆角卡片（16dp），渐变背景，与顶部徽标的黄绿蓝配色一致。

* 文案：标题（20–22sp，粗体）+ 副标题（12–14sp，浅灰）。默认文案：

  * 标题："艺术修图，一键出片"

  * 副标题："热门工具快捷直达"

* 快捷入口：3 个小按钮（魔法消除 / 拼图 / AI修图）。风格采用 FilledTonal 或 AssistChip 风格，圆角一致。

* 交互：按压缩放到 0.98，并淡背景高亮，松手恢复（与现有网格一致的按压规范）。

## 技术实现

* 新增 `@Composable fun MidHeroCard(onMagicEraseClick: () -> Unit, onCollageClick: () -> Unit, onFaceBeautyClick: () -> Unit)`。

* 布局：

  * `Box(modifier = Modifier.fillMaxWidth().height(180.dp).clip(RoundedCornerShape(16.dp)).background(Brush.radialGradient(listOf(Color(0xFFCCFF00), Color(0xFF00E1FF)))) .padding(16.dp))`

  * 内部 `Column`：`Text(标题)`、`Spacer(8.dp)`、`Text(副标题)`、`Spacer(12.dp)`、`Row` 三个按钮。

* 按压效果：在按钮外层使用 `MutableInteractionSource` + `collectIsPressedAsState()` 驱动 `scale(0.98f)` 与浅色 `background(Color(0x11000000))`。

## 接入位置

* 在 `HomeScreen` 的上半区（Header 与两排主按钮之后），在底部 `MenuGridSection` 之前插入 `MidHeroCard(...)`，保持上下分区结构不变。

* 按钮回调分别绑定到现有方法：

  * 魔法消除 → `onMagicEraseClick`

  * 拼图 → `onCollageClick`

  * AI修图 → `onFaceBeautyClick`

## 无障碍与性能

* 为按钮与卡片提供 `contentDescription`，文本对比度与字号可读。

* 纯 Compose 绘制，渐变与阴影静态，性能开销低；无额外依赖与 IO。

## 可选扩展（后续可开关）

* 卡片背景轻微“呼吸”透明度动画（`InfiniteTransition`），默认关闭。

* 根据系统主题/时间段动态调整渐变配色。
